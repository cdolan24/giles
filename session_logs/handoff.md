# Project Handoff Document

## Repository: giles

### Overview
This project is a modular web-scraper extension for the UC Library system, designed to provide fast, non-duplicative access to scientific literature and support expansion to other public digital libraries.

### Key Features
- Modular backend with MCP resource wrappers
- FastAPI server for search and metadata
- Advanced search filters
- Citation management and display
- Prerequisite recommender
- Ranking engine
- Entity linker for cross-source aggregation
- React frontend chat widget and filters
- Browser extension popup
- Comprehensive unit tests for backend modules
