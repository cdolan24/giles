# Chat Widget Design

This document outlines the design for the chat window interface, which will be implemented as a browser extension or web widget.

## Features
- User can ask questions about a field of study
- Display recommended papers, books, and prerequisite knowledge
- Show citations for all recommendations
- Allow follow-up questions and contextual answers

## UI Components
- Chat input box
- Message display area
- Citation display section
- Loading indicator

## Technologies
- React (recommended)
- Material-UI or custom CSS for styling
- WebSocket or REST API integration with backend

## Next Steps
- Scaffold React app in `frontend`
- Implement chat UI components
- Connect to backend API endpoints
